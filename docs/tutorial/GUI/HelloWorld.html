Hello World を作ってみよう

伝統のHello Worldをまず作ってみましょう。

import orto.ui.*;

public class HelloWorld {
  public HelloWorld() {
    Panel mainPanel = Browser.getMainWindow().getBody().createPanel();
    mainPanel.setHTML("Hello World");
    mainPanel.setVisible(true);
  }
}

簡単ですね。Ortoでは、public static void main(String[] args) を使いません。 あるクラスのインスタンス（オブジェクト）が作られることによりプログラムが開始します。 iアプリなどと同じです。

では、public HelloWorld() 内の３行を順番に見ていきましょう。

行番号説明
１ 表示するためのパネルを作っています。 パネルは物をのせるための透明のシートです。Java の Swing の JPanel と同じ考えです。 Browser.getMainWindow().getBody() で、HTML タグ内の BODY タグを取り出してきています。 その BODY タグに対して、createPanel() することにより、新しいパネルを作っています。
２ パネル内の文章を設定しています。 ここでは、Hello World としか、書いていませんが、HTML のタグを使うことも当然出来ます。
３ パネルを表示しています。 true なら表示、false なら非表示です。 準備している最中に表示すると汚らしくなるので、 createPanel() で作成したパネルは初期状態は非表示になっています。

コンパイル方法

javac -bootclasspath lib\ortoClasses.jar HelloWorld.java
java -jar bin\ortoar.jar -main HelloWorld HelloWorld.class

ここでは、 Orto の展開先に、HelloWorld.java を置いたとしましたが、 違う場合は、lib や bin のパスを適切に指定してください。
