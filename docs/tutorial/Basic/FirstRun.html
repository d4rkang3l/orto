動作環境を整えよう

Orto で Java のクラスファイルを HTML ファイルに変換するには、以下のものが必要です。

名前説明
Java2 SDK 1.4もしなければ、サン・マイクロシステムズのサイトからダウンロードしてインストールしてください。
Orto SDKOrtoのホームページからダウンロードしてください。

Javaソースファイルをコンパイルしよう

samplesに入っている、サルトラ算をコンパイルしてみましょう。現在のフォルダは、samplesとします。classes フォルダにはコンパイル済みのクラスがあります。

cd src
javac -bootclasspath ..\..\..\lib\ortoClasses.jar jp\orto\sample\monkeytiger\*.java

OrtoはJava2 Standard Edition の標準のクラスライブラリを使いません。 Java2 Micro Edition のクラスライブラリに近いのを使います。 そのため、-bootclasspath でortoClasses.jar を指定する必要があります。

HTML ファイルに変換しよう

MonkeyTiger.class から MonkeyTiger.html を作ります。

java -jar ..\..\..\bin\ortoar.jar -main jp.orto.sample.monkeytiger.MonkeyTiger -mainHTML ..\design.html jp\orto\sample\monkeytiger\MonkeyTiger.class

MonkeyTiger.htmlをブラウザで開けば実行します。 起動時に -main で指定したクラスのインスタンスオブジェクトが作られ、プログラムの実行が開始します。 これを指定することは必須です。 -jar の部分は、javaが処理し、-main の部分はortoar.jarが処理するので、必ず、-jar を先頭にしてください。
