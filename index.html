Orto で楽して、楽しもう！

Orto はプラグイン無しでブラウザ上で動く Java 風言語です。
何でプラグインがいらないって？それは、Dynamic HTML を利用しているからです。

もし、こんなことでお困りでしたら、ぜひお試しください。

  • Servlet/JSP, PHP のアプリは使い勝手やユーザインタフェイスが悪い
  • Servlet/JSP, PHP は開発効率が悪い

対応ブラウザ

Internet Explorer 4 以降、Netscape 6 以降、Mozilla、Opera 7 以降で動きます。
