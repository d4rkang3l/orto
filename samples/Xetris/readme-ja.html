Xetris

１．このサンプルについて

このサンプルは、いわゆる落ち物ゲームです。 Xetris-ja.html を開けば実行できます。

２．コンパイル方法

コンパイルするには、JDK 1.2 以降が必要です。

手動でコンパイルするには、以下のようにコマンドを入力してください。 現在のフォルダは、samplesとします。classes にはコンパイル済みのクラスがあります。

cd src
javac -bootclasspath ..\..\..\lib\ortoClasses.jar jp\orto\xetris\*.java

生成される３つのクラスファイルから Xetris-ja.html を作る必要があります。それは以下のようにやります。

java -jar ..\..\..\bin\ortoar.jar -main jp.orto.xetris.Xetris -mainHTML ..\design\design.html -encoding Shift_JIS jp\orto\xetris\*.class
copy Xetris.html ..\Xetris-ja.html

３．実行方法

Xetris-ja.html をブラウザから開くだけですが、 実行時に design フォルダを読みに行くので、Xetris-ja.html と design フォルダを 同じフォルダ内におく必要があります。
