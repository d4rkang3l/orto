みみず君

１．このサンプルについて

このサンプルはみみず君を成長させるゲームです。 Worm-ja.html を開けば実行できます。

２．コンパイル方法

コンパイルするには、JDK 1.2 以降が必要です。

手動でコンパイルするには、以下のようにコマンドを入力してください。 現在のフォルダは、samplesとします。classes にはコンパイル済みのクラスがあります。

cd src
javac -bootclasspath ..\..\..\lib\ortoClasses.jar jp\orto\worm\*.java

生成される３つのクラスファイルから Worm-ja.html を作る必要があります。それは以下のようにやります。

java -jar ..\..\..\bin\ortoar.jar -main jp.orto.worm.Worm -mainHTML ..\design\design.html -encoding Shift_JIS jp\orto\worm\*.class
copy Worm.html ..\Worm-ja.html
